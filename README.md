## For the end user

A complete clinical anchor running entirely in your browser. Patient records, appointment scheduling, clinical notes, prescribing, and CQC-aligned compliance tracking.

### Key rules

- **GDPR Article 9**: special category health data — explicit consent required
- **Caldicott Principles**: justify purpose, minimum necessary, need-to-know
- **CQC**: 5 key questions — Safe, Effective, Caring, Responsive, Well-led
- **Duty of candour**: HSCA 2008 Regulation 20
- **Record retention**: 7 years (NHS standard)
